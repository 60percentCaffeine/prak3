Программа включает три процесса: отца, сына и внука. Между ними циркулирует маркер - массив типа int длины 2. Изначально нулевой элемент равен 0, первый равен 1. Изначально маркер находится у внука. Внук вычисляет s - сумму элементов массива. Затем в 0й элемент массива копирует 1й, а в 1й элемент записывает s. Затем маркер пересылается сыну. Сын выполняет те же действия над маркером, пересылает отцу. Тот делает то же самое, отправляет маркер внуку и так по кругу. Таким образом, маршрут передачи маркера следующий: внук->сын->отец->внук.

Циркуляция маркера продолжается до тех пор, пока 1й элемент массива не превысит некоторое  пороговое значение (которое передаётся через параметры командной строки). Для коммуникаций  использовать ровно один неименованный канал.

Синхронизацию сделать на сигналах либо на семафорах.
